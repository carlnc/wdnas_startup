Instructions:

  1. SSH in
  2. Modify /mnt/HD/HD_a2/Nas_Prog/Startup/config/settings.sh
  3. Set the contents to 
    STARTUP_ENTRY_SCRIPT=/path/to/your/startup/script.sh
  4. Ensure your script is executable: 
    chmod a+x /path/to/your/startup/script.sh
  5. Reboot

Testing:

  • If you want to check your script executes, run: 
    /mnt/HD/HD_a2/Nas_Prog/Startup/start.sh
